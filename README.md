# Jon Kost — Portfolio Placeholder

A minimal coming soon page for [jonkost.com](https://jonkost.com).

## What it is

Single-page HTML placeholder while the full portfolio site is in development. Dark background, fade-in animations, fully responsive.

## Usage

Drop `index.html` into your root directory or hosting platform of choice. No dependencies, no build step — it just works.

## Hosting options

- **Netlify** — drag and drop the file at netlify.com/drop
- **GitHub Pages** — push to a repo and enable Pages in settings
- **Squarespace / Webflow** — use as a custom code embed or replace index
- **Any static host** — it's one file

## Customization

All editable content is in the HTML directly:

- **Name** — `<h1>Jon Kost</h1>`
- **Subtitle** — `<p class="subtitle">Creator &nbsp;&nbsp; Producer &nbsp;&nbsp; Educator</p>`
- **Bio** — `<p class="bio">...</p>`
- **Status pill** — `<div class="status">...</div>`
- **Colors** — edit the CSS variables at the top of the `<style>` block

## Stack

Plain HTML, CSS, no JavaScript frameworks, no external dependencies.
